# Environments

* `Always respect your environments`

# How To

To build, use the following commands

```
./configure
./make clean all
```

# Targets

The following are the supported targets for the build

* `linux`, `x86`
* `linux`, `x64`
* `linux`, `arm`
* `osx`, `x86`
* `osx`, `x64`

# Install

To install, run the following command

```
./make install
```

# Uninstall

To uninstall, run the following command

```
./make uninstall
```
